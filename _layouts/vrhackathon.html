A VR app using Unity game engine was developed by me as a part of an online hackathon aimed at creating an interactive solution for NFT viewing and bidding via virtual reality.

Proposed Idea

The prototype demonstrate a virutal reality room where one has the opportunity to view all of the possible events that resides in the blockchain. The room has the capability to request for information from the backend system and present these information such as images, videos and text stored in the room. They also have the option to bid on a specific NFT from the virtual reality application.

Tools & Softwares Used

Unity Engine, and Oculus Quest

Team Information

  • Rahul - Prototype development Using Unity Engine (C#) and testing the final product on Oculus Quest 2.

Project Description

The project was setup completely in Unity Engine and it was designated to run on Oculus quest devices. The development spanned over a period of 10 days - I was mostly working on it after my work hours. The aim was to create an interactive environment that allows users to view and bid on various available NFT's from virtual reality. There was a web version that allows one to perform the same but I was approached by a friend of mine to create a VR version so that they could offer an interactive version of the same product.



This was just a prototype that was designed to facilitate all of the requirements in a short of span of time due to the hackathon time constraints.
